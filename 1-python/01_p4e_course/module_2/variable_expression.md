## Section 1: Conceptual Questions (with answers)

### Question 1
**What is the difference between an operator and an expression in Python?**

**Answer:**
- Operator: A symbol that performs an action on operands (e.g., +, -, *, /)
- Expression: A combination of values, variables, and operators that produces a result (e.g., 5 + 3) ✅

---

### Question 2
**What does the input() function do in Python?**

**Answer:**
- Takes data from the user via the keyboard  
- Always returns the data as a string ✅

---

### Question 3
**Why do we need type conversion when working with user input?**

**Answer:**
- Because input() returns a string  
- We must convert it using int() or float() to perform calculations ✅

---

### Question 4
**Without looking at notes — what are two ways to convert a string to a number in Python?**

**Answer:**
- int() → converts to whole numbers  
- float() → converts to decimal numbers ✅
