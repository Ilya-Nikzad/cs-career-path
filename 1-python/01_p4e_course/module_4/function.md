# Module 4: Functions in Python

## Pseudo-code Example
BEGIN PROGRAM  
    FUNCTION greeting(name)  
        RETURN "Hello, " + name + "!"  
    END FUNCTION  

    PRINT greeting("Hanna")  
END PROGRAM  

---

## Conceptual Questions & Answers

**Q1: Difference between parameter and argument**  
A parameter is a variable defined in the function.  
An argument is the actual value passed into the function.

**Q2: What does `return` do?**  
It stops the function and sends a value back to the caller.

**Q3: What happens if you use only `print()` instead of `return`?**  
It will display output but the function will return `None` by default.

**Q4: Two main purposes of functions in Python**  
- Reusability: reuse code multiple times  
- Readability: makes code cleaner and easier to understand
