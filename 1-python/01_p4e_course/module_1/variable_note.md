## 📘 Section 1: Conceptual Questions (with answers)

### ❓ Question 1: What does the `print()` function do in Python?
**Answer:** The `print()` function displays text or values on the screen. ✅

---

### ❓ Question 2: What is a variable in Python, and why do we use it?
**Answer:** A variable is a reusable placeholder for a value that can change later. ✅

---

### ❓ Question 3: Name at least three common data types in Python.
**Answer:**  
- `int` – whole numbers  
- `float` – decimal numbers  
- `str` – text (strings)  

---

### ❓ Question 4: What happens if you forget to convert a number to a string when concatenating it with text?
**Answer:** Python gives a `TypeError` because it cannot directly concatenate a string (`str`) and an integer (`int`). ✅

## 💡 Tips / Best Practices

- Always write pseudo-code first before coding
- Convert numbers to strings when combining with text using `str()`
- Keep variable names meaningful
- Test code step by step to catch errors early
