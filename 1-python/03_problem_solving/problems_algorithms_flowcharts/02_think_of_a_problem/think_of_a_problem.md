# 🟠 Algorithm Task

## 👉 Decide if someone can enter a club

---

### 📄 Original Content


# ALGORITHM TASK

Write steps for:

👉 “Decide if someone can enter a club”

Rules:

Must be age 18+

# ALGORITHM ANSWER

Input: Person’s age

Output: “Allowed” or “Not allowed”

Steps:

Ask for the person’s age.
Check if the age is a number.
Compare the age with 18.
If age is 18 or greater, then:
Output “Allowed to enter the club”.
If age is less than 18, then:
Output “Not allowed to enter the club”.
End the process.
