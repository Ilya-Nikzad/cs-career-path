Summary Notes: Algorithms and Their History
1. What is an algorithm?
An algorithm is a step-by-step set of instructions used to solve a problem.
It does not depend on computers and existed long before digital machines.
Anyone can execute an algorithm manually, although it may be slow or repetitive.
2. Historical background
The concept of algorithms predates modern computing by hundreds of years.
Muhammad ibn Musa al-Khwarizmi (Al-Khwarizmi), a Persian scholar, is considered a key figure in the development of algebra.
His work “The Compendious Book on Calculation by Completion and Balancing” introduced systematic methods for solving linear and quadratic equations.
The word algorithm comes from the Latinized form of his name (algorithmi).
Earlier still, Euclid (c. 300 BC) developed an algorithm for finding the greatest common divisor (GCD), now known as Euclid’s algorithm.
3. Algorithms vs computers
Algorithms are abstract mathematical ideas.
A computer program is a practical implementation of an algorithm using a programming language.
Computers are useful because they execute repetitive steps quickly and reliably, unlike humans.
4. Key relationship
Algorithm → abstract idea (step-by-step method)
Program → implementation of an algorithm in code
Computer → machine that executes programs efficiently
5. Main takeaway
Algorithms are general, timeless concepts independent of technology.
Modern computing is based on translating these abstract methods into programming languages that computers can understand and execute.
