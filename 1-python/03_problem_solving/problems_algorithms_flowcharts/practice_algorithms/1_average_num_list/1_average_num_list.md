# Algorithm: Find Average of Numbers in a List
1. BEGIN  
2. Initialize a list of numbers (Input / Collection)  
3. Set sum = 0 (Memory / Initialization)  
4. Set count = 0 (Memory / Initialization)  
5. Loop through each number in the list (Repetition)  
6. Add each number to sum (Computation)  
7. Increase count by 1 (Computation / Counter Update)  
8. Compute average = sum ÷ count (Computation)  
9. Display the average (Output)  
10. END
