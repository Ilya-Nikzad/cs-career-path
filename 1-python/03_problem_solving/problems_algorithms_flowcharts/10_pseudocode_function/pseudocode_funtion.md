# 1: What is a Function?
A function is a reusable block of logic that:
takes an input
performs steps (body)
returns an output
It solves a specific problem.

Key idea
Function = “question solver”

# 2: Structure of a Function in Pseudocode
A function in pseudocode has a fixed structure:
Start with function
Name of function
Input inside brackets
Body logic steps
End with end function

Key idea
Function is a self-contained decision machine

# 3: What does mod 2 actually mean?
mod means remainder after division.
So:
n mod 2 asks: what is the remainder when dividing by 2?

What this tells us
If remainder = 0 → number is even
If remainder = 1 → number is odd

# 4: What does return do in a function?
The keyword return:
sends output back to the caller
immediately stops the function
Nothing after return runs.

Key idea
return = output + exit function instantly

# Q1: What are the 3 main parts of a function in pseudocode?
function declaration (name + input)
body (steps/logic)
end function

# SUMMARY
What you learned:
Function = input → process → output
Pseudocode structure = function + body + end function
mod 2 checks even/odd
return outputs value + stops function
Execution follows step-by-step logic
Always trace carefully in exams
