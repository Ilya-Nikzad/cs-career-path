# 📘 Chapter 8 – Key Concept Notes (Lists & Collections)

## 🎯 Core Idea
You are moving from:

- 👉 Single-value thinking  
- 👉 Multi-value thinking (collections)
### Before:
👉 One variable → one value

### Now:
👉 One variable → multiple values

## 🧠 Algorithm vs Data Structure

- 👉 Algorithm = steps (what to do)  
- 👉 Data Structure = how data is organized  
## 🧠 Key Rule

👉 Square brackets represent a **list**

Question 1: What symbol is used to create a list in Python?
Using square brackets symbol

Question 2: What will be stored in this variable?
#data = [10, 20, 30]
data is a list that stores multiple values

Question 3: Can a list store different types of data?
Yes, a list in Python can store different data types

Question 4: What is the main difference between: list and a vairable 
A variable stores one value, while a list stores multiple values in one structure.

Task 1: Find the sum of a list of numbers
BEGIN PROGRAM
1. Initialize a list of numbers
2. Set total = 0
3. Loop through each number in the list:
      add number to total
4. Print total
END PROGRAM

Task 2: Find the maximum number in a list
BEGIN PROGRAM
1. Initialize a list of numbers
2. Set max = first element of the list
3. Loop through remaining numbers in the list:
      if number > max:
          update max = number
4. Print max
END PROGRAM

Task 3: “Count how many numbers are EVEN in a list”
BEGIN PROGRAM
1. Initialize a list of numbers
2. Set count = 0
3. Loop through each number in the list:
      if number % 2 == 0:
          increase count by 1
4. Print count
END PROGRAM
