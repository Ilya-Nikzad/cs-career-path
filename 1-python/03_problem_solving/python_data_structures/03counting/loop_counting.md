🧾 Algorithm Steps
1. Start

Begin the process.

2. Input

Take a word (example: "book").

3. Initialize Counter

Set counter to 0.

4. Traverse the Word

Look at each letter in the word one by one.

5. Counting Process

For each letter, increase the counter by 1.

6. Repeat

Continue the process until all letters in the word have been checked.

7. Output

Display the final value of the counter (total number of letters).

8. End

Stop the process.

📊 Example

Input: "book"

Process:

b → counter becomes 1
o → counter becomes 2
o → counter becomes 3
k → counter becomes 4

Output: 4

🧠 Summary

This algorithm counts how many characters are in a word by checking each letter one by one and increasing a counter.
