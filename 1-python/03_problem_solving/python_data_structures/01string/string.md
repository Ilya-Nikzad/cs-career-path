# Python Strings – Notes

## Q1: What is a string in Python? Give an example.
A string is a sequence of characters inside quotes.  
Example: "Python"

---

## Q2: How do you read input from the user in Python, and what data type does it return?
We use input() to read user input.  
It always returns a string (str) by default.

---

## Q3: What does it mean to convert a string in Python? Give one example.
It means changing the data type (such as to int or float) so we can perform arithmetic operations.  
Example: converting "10" into an integer.

---

## Q4: What does indexing mean in a string? What is the index of the first character?
Indexing means accessing a character in a string using its position.  
The first character always has index 0.

---

## Q5: What happens if the index is bigger than the string length?
If the index is greater than or equal to the length of the string, Python gives an IndexError.

---

## Q6: What does the len() function do in Python when used on a string?
It returns the number of characters in a string.
