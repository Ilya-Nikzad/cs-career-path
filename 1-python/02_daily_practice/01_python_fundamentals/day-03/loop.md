# Python Loops Notes (for & while)

---

## 🔵 FOR LOOP

### 1. For loop
A `for` loop repeats code a fixed number of times.

---

### 2. range()
`range(5)` gives numbers from 0 to 4  
(5 is not included)

---

### 3. Pattern printing
`"*" * i` repeats a character i times  
Used to create patterns like triangles

---

### 4. Nested loop
A loop inside another loop.  
The inner loop finishes completely for every single outer loop cycle.

---

### 5. end=""
Keeps output on the same line (no newline after print)

---

### 6. print()
Moves output to the next line automatically

---

## 🟢 WHILE LOOP

### 1. While loop
A `while` loop repeats as long as a condition is TRUE.

---

### 2. Validation loop
A loop that repeats until the user enters correct input.

---

## 🔁 KEY DIFFERENCE

| Loop type | Use case |
|----------|----------|
| for loop  | Fixed number of repeats |
| while loop | Unknown repeats (until condition becomes false) |

---

## 💡 EXTRA IMPORTANT CONCEPTS

### break
Stops the loop immediately.

### continue
Skips the current iteration and goes to the next one.
