# 📘 Python Functions – Clean Notes

## 🔹 Function
A function is a named block of code that can be reused to perform a specific task.

---

## 🔹 Return Value
- A function in Python always returns something.
- If no `return` statement is used, Python automatically returns `None`.
- `return` is used when you want to send a value back from the function so it can be used later.

👉 Without `return`, a function can still run and perform actions (like printing), but it cannot provide a reusable result.

---

## 🔹 Parameters
Parameters are variables defined in a function.  
They allow a function to accept input values and work with different data each time it is called.

---

## 🔹 Arguments
Arguments are the actual values passed into a function when it is called.

---

## 🔑 Quick Summary
- Function = reusable block of code  
- Parameter = input variable inside function  
- Argument = real value given to function  
- return = sends back a usable result  
- No return = function returns `None`
