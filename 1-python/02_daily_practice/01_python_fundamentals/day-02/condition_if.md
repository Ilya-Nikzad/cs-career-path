# Python Practice 1: if / else Basics

## 1. What is if / else?
In Python, `if` and `else` are used to make decisions.

- `if` = runs code when condition is True  
- `else` = runs code when condition is False  

---

## 2. Comparison Operators

| Operator | Meaning | Example |
|----------|--------|---------|
| == | equal to | a == b |
| != | not equal to | a != b |
| > | greater than | a > b |
| < | less than | a < b |
| >= | greater or equal | a >= b |
| <= | less or equal | a <= b |
