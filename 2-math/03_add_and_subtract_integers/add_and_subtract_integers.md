## Chunk 2: Absolute Value (Distance from Zero)

Absolute value tells us how far a number is from 0 on the number line.

🔹 Notation  
We write absolute value like this:
|n|  
This means: “distance of n from 0”

Absolute value means how far a number is from zero,  
without caring about direction (positive or negative).

So:
|5| = 5  
|-5| = 5  

Because both are 5 steps away from 0. distance always positive

## Chunk 3: Adding Integers (Same Signs)

When we add integers, the rule depends on the signs.

🔹 Case 1: Same signs (both positive or both negative)

If signs are the same:
Add the numbers normally  
Keep the same sign  

🔹 Example 1 (positive + positive)
5 + 3 = 8  
Both are positive → result is positive  

🔹 Example 2 (negative + negative)
(−5) + (−3) = −8  
Both are negative → result stays negative  

🔹 Key idea
Same signs → add values → keep the sign  

---

## Chunk 4: Adding Integers (Different Signs)

Now we look at what happens when signs are different.

🔹 Case 2: Different signs (one positive, one negative)

When signs are different:
Subtract the smaller number from the larger number  
Keep the sign of the larger number  

🔹 Example 1
(−5) + 3  

Step:
5 − 3 = 2  
Bigger number is 5 (negative) → answer is −2  

🔹 Example 2
5 + (−3)  

Step:
5 − 3 = 2  
Bigger number is 5 (positive) → answer is +2  

🔹 Key idea
Different signs → subtract → keep sign of bigger number

## Chunk 5: Subtracting Integers

Subtraction with integers can look confusing at first, but there is one powerful rule.

🔹 Main rule
Change subtraction into addition of the opposite.

This means:
keep the first number  
change “−” into “+”  
flip the sign of the second number  

🔹 Example 1
5 − 3  

Becomes:
5 + (−3)  

Now use addition rules:
answer = 2  

🔹 Example 2
5 − (−3)  

Becomes:
5 + 3  

Answer:
8  

🔹 Example 3
(−5) − 3  

Becomes:
(−5) + (−3)  

Answer:
−8
